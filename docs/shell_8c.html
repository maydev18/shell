Creates a command for the pipeline operator (|) from the provided arguments.

This function processes the arguments and separates a command to be executed in a pipeline. It stores the individual command in the cmnd[] array.

Parameters
argsThe array of arguments (sanitized input from stdin).
cmndThe array to store a single command extracted from args[].
current_argThe current index in args[] being processed.
Returns
The updated value of current_arg to be used for the next command.
Note
This function modifies the cmnd[] array to store the extracted command.

void pipe_command_handler ( char *  args[])

Executes the command being made by create_command_for_pipe()

This function extracts the next command to be run from args and then fork a process to execute that command. As we are using this function for pipe(|), hence the output from the prevous command is passed as an input to the last command, except two exceptions * the first command's input remains stdin and the last command's output remains stdout. pipe() system call is being used to create pipes between 2 processes to read and write from.

argsThe array of arguments (sanitized input from stdin).
